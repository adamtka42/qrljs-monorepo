RLP | @theqrl/rlp

Basic browser usage of the generic RLP helpers retained for the QRL local execution stack.

Run the Example

  1. Go to packages/rlp.
  2. Build the package with npm run build.
  3. Start Vite with npx vite.
  4. Open http://localhost:5173/examples/browser.html.
  5. Open the browser console and check the example output.
